rsync 是一个强大的文件同步和传输工具，通常用于在本地和远程主机之间高效地复制和同步文件。以下是 rsync 的一些基本用法和示例：

基本语法
sh
rsync [选项] 来源 目标
常用选项
-a 或 --archive: 归档模式，递归复制并保留符号链接、权限、时间戳等。
-v 或 --verbose: 显示详细输出信息。
-z 或 --compress: 在传输时压缩文件数据，以减少带宽使用。
-P: 等同于 --partial --progress，显示进度并保留部分传输的文件，以便中断后继续。
-r 或 --recursive: 递归复制目录和子目录。
-e ssh 或 --rsh=ssh: 使用 SSH 进行远程传输。
